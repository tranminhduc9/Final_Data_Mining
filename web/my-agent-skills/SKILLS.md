# Hướng dẫn quy trình phát triển TechPulse VN

Kỹ năng này cung cấp các công cụ để làm việc với dự án. Khi thực hiện các tác vụ liên quan đến setup hoặc viết code, hãy tham khảo các thư mục con:

- **Chạy script:** Mở thư mục `scripts/` để dùng các lệnh tự động hóa (cài đặt, khởi chạy backend và frontend).
- **Đọc tài liệu:** Mở thư mục `references/` để hiểu quy tắc viết code (convention), cấu trúc dự án và luồng dữ liệu Graph RAG.
- **Dùng template:** Mở thư mục `assets/` để lấy các mẫu file cấu hình hoặc component có sẵn.
